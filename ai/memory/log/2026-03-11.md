## MD Style Guide Added

**File**: `ai/core/base-instructions.md`

A new `## MD Style Guide` section was added at the end of the file with 8 rules:

- **MD025 / single-h1** ‚Äî one `#` per file
- **MD022 / blanks-around-headings** ‚Äî blank lines before/after all headings
- **MD032 / blanks-around-lists** ‚Äî blank lines before/after list blocks
- **MD031 / blanks-around-fences** ‚Äî blank lines before/after code fences
- **MD012 / no-multiple-blanks** ‚Äî no consecutive blank lines
- **MD009 / no-trailing-spaces** ‚Äî no trailing whitespace
- **MD010 / no-hard-tabs** ‚Äî use spaces not tabs
- **MD041 / first-line-heading** ‚Äî first line of every file must be `#` H1

This list is intended to grow over time as the user expands the whitelist.
